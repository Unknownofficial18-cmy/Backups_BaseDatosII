Comando para hacer un backup en postgres(cambiar nombre de contenedor, nombre de base de datos y ruta):

sudo docker exec -t postgres_bd pg_dump -U postgres -d clinica_postgres > /mnt/c/backups/clinica_postgres_backup.sql

Comando para hacer la restauraci√≥n de postgres(Ubuntu):
Cambiar ruta (cat),nombre del contenedor, y que exista primero la base de datos:


cat /mnt/c/backups/clinica_postgres_backup.sql | sudo docker exec -i postgres_bd psql -U postgres -d clinica_postgresrestaurar


Comando para hacer backup en MySQL:

sudo docker exec -t <nombre_contenedor_mysql> mysqldump -u <usuario> -p<password> <nombre_base_datos> > <ruta_en_host>/<archivo_backup>.sql

Comando para restaurar:
cat /mnt/c/backups/clinica_mysql_backup.sql | sudo docker exec -i mysql_server mysql -u root -pMiPassword clinica_mysql
